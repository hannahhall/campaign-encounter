# Dungeons and Dragons Encounter Tracker

## To run the server:
1. `cd api`
2. `python manage.py migrate`
3. `python manage.py runserver`
4. The server is running on port 8000

## To run the client
1. `cd ghosts-of-saltmarsh`
2. `npm start`
3. The client is running on port 4200

## Try it out
http://localhost:4200
